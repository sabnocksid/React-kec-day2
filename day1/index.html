HTML Recap

HTML(HyperText Markup Language) is not a programming language.
It is a scripting language.

HTML document is made up of html elements.

There are two types of html tags
  1. Paired tags
  2. Unpaired/Empty tags

Some common html tags:

  • paragraph tag <p> : paired

  • img tag: unpaired
  • heading tags: paired
  • div: paired

Some tags are semantic. They have special meaning but might not have visible effects. Like: main, article, div, section etc.
